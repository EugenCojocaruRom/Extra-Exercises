EX_26MAR - Most Stocked Product
Create a method named 'printMostStockedProduct' that takes a HashMap<String, Integer> (where keys are product names and values are quantities) as input and performs the following operations:
 1. Iterate over the inventory using 'entrySet()' to find the product with the highest quantity.
 2. Print the product with the highest quantity in the format:
	Most Stocked Product: <product name>, Quantity: <quantity>
 3. If the inventory is empty, print:
	No products in inventory.
