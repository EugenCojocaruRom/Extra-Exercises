EX\_21MAR - HashMap exercise 6

Create a method named modifyAndResetHashMap that takes a HashMap<String, Integer> named data as input and performs the following operations:

1\. Check if the HashMap is empty and print:

&#x09;Is empty: true  

&#x09;or

&#x09;Is empty: false  

2\. Print the size of the HashMap:

&#x09;Size: X

&#x09;(where X is the number of key-value pairs in data)

3\. Add a new key-value pair ("Reset", 0) to the HashMap.

&#x20;  \* If the key "Reset" already exists, do not modify it.

&#x20;  \* Print the updated size of the HashMap:

&#x09;Size after modification: Y  

&#x09;(where Y is the new size)

4\. Clear the HashMap and check if it’s empty again. Print:

&#x09;Is empty after clear: true
