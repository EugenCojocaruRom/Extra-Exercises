EX\_06FEB - Gradebook analyzer

Create a gradebook analyzer with three methods that perform different array calculations:

* Create method getClassStats that:

&nbsp;- Takes an array of grades (integers)

&nbsp;- Returns a string with class statistics in this format:

&nbsp;   Class Average: ...(double)

&nbsp;   Highest Grade: ...(int)

&nbsp;   Lowest Grade: ...(int)

&nbsp;   Passing Grades: x/y (int/int)

&nbsp;   \*(Passing grade is 60 or above)

* Create method getLetterGrades that:

&nbsp;- Takes an array of grades (integers)

&nbsp;- Returns an array with count of each letter grade in this format:

&nbsp;   A (90-100): ...

&nbsp;   B (80-89): ...

&nbsp;   C (70-79): ...

&nbsp;   D (60-69): ...

&nbsp;   F (0-59): ...

* Create method getImprovement that:

&nbsp;- Takes two arrays: starting grades and ending grades

&nbsp;- Returns improvement statistics in this format:

&nbsp;   Average Improvement: ... points (double)

&nbsp;   Most Improved: ... points (int)

&nbsp;   Students Improved: x/y (int/int)
