Codecademy - ex08DEC - FizzBuzz

FizzBuzz exercise - given a number of elements (enter a number from the keyboard), replace the elements that are multiple of 3 with 'Fizz',

the multiples of 5 with 'Buzz' and the multiples of both 3 and 5 with 'FizzBuzz'. Print the list containing the elements replaced with the words above.
