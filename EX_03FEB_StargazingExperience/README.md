EX\_03FEB - Stargazing experience

Create a function named countStars that receives initialStars and minutes as its parameters. For each minute that passes, one more star is seen.

Parameters:

&nbsp;- initialStars (int): The number of stars visible at the beginning of the observation.

&nbsp;- minutes (int): The number of minutes spent stargazing.

The function returns a formatted string, which includes:

 - The initial number of stars

 - The final number of stars

 - The number of minutes spent stargazing

 - A simple ASCII art representation of stars (use asterisks \*)
