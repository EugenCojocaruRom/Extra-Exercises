EX\_11FEB - Student report card

Create a student report card system, with the following inputs: studentName, studentId, mathGrade, scienceGrade, englishGrade, attendance.

Create three formatted strings:

\* headerInfo: Format student information:

&nbsp;- Name -> capitalized (first letter of each word)

&nbsp;- ID -> padded with zeros to 5 digits format: e.g. "Student: Surname Name, ID: 00123"

\* gradesInfo: Format grades:

&nbsp;- All grades must show 2 decimal places Format: e.g. "Math: 92.56, Science: 88.78, English: 95.11"

\* attendanceInfo: Format attendance:

&nbsp;- Attendance shown as percentage with 1 decimal place format: e.g. "Attendance: 85.2%"
