EX\_10MAR - 2D Array Patterns

Create a method named 'printAdvancedPatterns' that takes a square 2D array of integers (matrix) as input and prints the following patterns:

1\. Both Diagonals Together:

&nbsp;- Print both the main diagonal and anti-diagonal in a single traversal.

&nbsp;- First, print all elements of the main diagonal (top-left to bottom-right), then print the elements of the anti-diagonal (top-right to bottom-left).

&nbsp;- If the matrix has an odd size, the center element belongs to both diagonals — print it only once (as part of the main diagonal).

2\. Spiral Order Traversal:

&nbsp;- Print the matrix elements in a spiral order starting from the top-left corner.
