EX\_05FEB - Arrays operations

Create a method named 'calculateStats' that takes an array of integers as input and performs the following operations:

&nbsp;- Calculates the sum of all elements in the array.

&nbsp;- Calculates the average of the elements in the array.

&nbsp;- Finds the maximum element in the array.

&nbsp;- Finds the minimum element in the array.

The method should return an array of doubles containing the sum, average, maximum, and minimum, in that order.
