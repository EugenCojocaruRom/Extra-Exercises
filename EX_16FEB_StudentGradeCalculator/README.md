EX\_16FEB - Student grade calculator

Create a method named calculateAverageGrade that takes an array of integers (representing student grades) as input and returns the average grade as a double.

The method should do the following:

&nbsp;- Calculate the sum of all grades in the array.

&nbsp;- Calculate the average grade by dividing the sum by the number of grades.

&nbsp;- Return the average grade

&nbsp;- Print the result using the following format: Average grade: \[The result] and format the result to two decimal places.
