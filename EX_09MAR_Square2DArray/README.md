EX\_09MAR - Square 2D Array

Create a method named 'printPatterns' that takes a square 2D array of integers (matrix) as input and prints the following patterns:

&nbsp;- Main Diagonal: Print all elements where the row index equals the column index.

&nbsp;- Anti-Diagonal: Print all elements where the sum of the row and column indices equals the size of the matrix minus 1.

&nbsp;- Borders: Print the elements of the top, bottom, left, and right borders of the matrix.
