EX\_31JAN - Student's grades

Create a program that manages a student's grades. Create three methods that modify arrays in different ways:

* Create method improveGrade that:

 - Takes an array of integers and an index

 - Increases the grade at that index by 5 points

 - If the new grade would be over 100, set it to 100

* Create method replaceFailingGrade that:

 - Takes an array of integers

 - Replaces any grade below 60 with 60

* Create method addBonusPoints that:

 - Takes an array of integers

 - Adds 2 points to all grades

Print the array after each modification
