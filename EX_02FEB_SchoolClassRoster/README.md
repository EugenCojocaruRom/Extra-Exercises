EX\_02FEB - School class roster

Create a program that manages a school class roster. Create three methods that use different array operations:

* Create method createSeatingChart that:

&nbsp;- Takes two arrays: student names and their grades

&nbsp;- Returns a new sorted array combining both in format "Name: Grade"

* Create method findTopStudents that:

 - Takes an array of integers (grades)

 - Creates and returns a new array filled with "A+" for grades 90 and above, "A" for others

* Create method isClassEqual that:

 - Takes two arrays of grades

 - Returns true if arrays have exactly same grades (order matters)
