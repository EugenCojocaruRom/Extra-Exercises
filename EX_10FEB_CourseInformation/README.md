EX\_10FEB - Course information

Create a program that processes course information. Create three methods that handle different aspects of string processing:

1\. Create method formatCourseInfo that:

&nbsp;  \* Takes a course string like "MATH101:Algebra:Monday,Wednesday:9AM"

&nbsp;  \* Splits it by ":" and returns formatted string

2\. Create method createCourseList that:

&nbsp;  \* Takes an array of course codes: \["MATH101", "ENG202", "HIST101"]

&nbsp;  \* Creates a comma-separated list

3\. Create method formatSchedule that:

&nbsp;  \* Takes days string: "Monday,Wednesday,Friday"

&nbsp;  \* Splits by comma and joins with " | "
