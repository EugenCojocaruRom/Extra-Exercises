EX\_20JAN - Sort and count words

Receive a text (as a String) and:

&nbsp;- split it into separate words (using spaces as separators) and return all words as a list (preserving their initial order, and including possible duplicates)

&nbsp;- return the number of words it contains

&nbsp;- return all the words contained in the text, sorted alphabetically

&nbsp;- return only the distinct words (no duplicates) from the text, in their initial order
