EX\_14FEB - Scorecard message

Create a function named 'generateBowlingMessage' that receives playerName and score as its parameters.

Return a message with three parts: a greeting with the player's name, their score, and a comment based on performance.

Parameters:

&nbsp;- playerName (String): The name of the player.

&nbsp;- score (int): The player's bowling score (0-300).

Use the following performance comments based on the score:

&nbsp;- Score 0-50: "Keep practicing!"

&nbsp;- Score 51-100: "Nice effort!"

&nbsp;- Score 101-150: "Great job!"

&nbsp;- Score 151-200: "Wow, you're a natural!"

&nbsp;- Score 201-300: "Perfect game! You're a bowling champion!"

Handle any leading or trailing whitespace in the player's name.
