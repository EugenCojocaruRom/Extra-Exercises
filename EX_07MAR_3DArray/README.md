EX\_07MAR - 3D Array

Write a program that gets that takes three integer arguments: depth, rows, and cols.

Create a 3D array of integers with the specified dimensions.

Initialize each element of the array with the product of its depth, row, and column indices (starting from 1).

For example, the element at \[d]\[r]\[c] should have the value (d + 1) \* (r + 1) \* (c + 1).

Finally print the resulting 3D array to the console using nested loops.
