EX\_27JAN - Areas of shapes

Calculate the area of different shapes using method overloading. Create four versions of a method called 'calculateArea':

 - Method that calculates area of a square:

        Takes one integer parameter (side length)

        Returns the area as an integer

 - Method that calculates area of a rectangle:

        Takes two integer parameters (length and width)

        Returns the area as an integer

 - Method that calculates area of a triangle:

        Takes one integer parameter (length) and one double parameter (height)

        Returns the area as a double

 	Trim to display only 2 decimal points

 - Method that calculates area of a circle:

        Takes one double parameter (radius)

        Returns the area as a double (use 3.14 for pi)

 	Trim to display only 2 decimal points
