EX\_07FEB - Username checker

Create a username checker with three methods that use String methods:

* Create method checkLength that:

&nbsp;- Takes a username String

&nbsp;- Returns message in format:

&nbsp;   Username length: ... (int)

&nbsp;   Long enough: ... (boolean)

&nbsp;   \*(Long enough means length >= 6)

* Create method checkStart that:

&nbsp;- Takes a username String

&nbsp;- Checks if it starts with "user\_" or "admin\_"

&nbsp;- Returns message in format:

&nbsp;   Starts with user\_: ... (boolean)

&nbsp;   Starts with admin\_: ... (boolean)

* Create method modifyUsername that:

&nbsp;- Takes a username String

&nbsp;- Gets first 5 characters using substring

&nbsp;- Converts to uppercase

&nbsp;- Returns message in format:

&nbsp;   First 5 chars: ... (String)

&nbsp;   Uppercase: ... (String.toUpperCase())
