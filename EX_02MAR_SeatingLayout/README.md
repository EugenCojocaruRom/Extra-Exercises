EX\_02MAR - Seating Layout

Write a function 'generateFestivalSeating' that takes rows, cols and returns a String\[]\[] representing seating labels.

Each seat label shows its position in format "R\[row]C\[col]-\[row\_binary]|\[col\_binary]" where row and column numbers are converted to binary.

Parameters:

&nbsp;- rows (int): Number of rows in the seating area

&nbsp;- cols (int): Number of columns in the seating area

Returns: 2D string array with seat labels. Format: "R1C2-1|10"
