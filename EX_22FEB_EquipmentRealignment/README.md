EX\_22FEB - Equipment Realignment

Create a function named 'realignEquipment' that receives 'equipment' and 'increment' as its parameters.

The function should perform the following operations:

&nbsp;- Increment each element in the equipment array by the value of increment.

&nbsp;- Reverse the order of the rows in the equipment array.

Parameters:

&nbsp;- equipment (int\[]\[]): A 2D array of integers representing the current settings of the equipment. Each row represents a different piece of equipment, and each column represents a specific setting.

&nbsp;- increment (int): An integer value by which each setting should be increased. This value can be positive or negative.

The function returns a 2D integer array representing the realigned equipment settings after performing the required operations.

Constraints:

&nbsp;- The number of rows in the equipment array will be between 1 and 100.

&nbsp;- The number of columns in the equipment array will be between 1 and 100.

&nbsp;- The increment value will be between -100 and 100.
