EX\_20MAR - Daily Schedule Formatter

Create a function named createRanchSchedule that receives morningTasks and eveningTasks as its parameters.

The function should combine the morning and evening tasks into a formatted daily schedule.

Follow these steps to create the schedule:

&#x20;- Concatenate the morningTasks and eveningTasks arrays.

&#x20;- Create a formatted string representing the daily schedule.

&#x20;- Number each task and prefix it with either "Morning:" or "Evening:" based on its original array.

&#x20;- Add a header and footer to the schedule.

Parameters:

&#x20;- morningTasks (String\[]): An array of strings representing morning tasks.

&#x20;- eveningTasks (String\[]): An array of strings representing evening tasks.

The function returns a String containing the formatted daily schedule.
