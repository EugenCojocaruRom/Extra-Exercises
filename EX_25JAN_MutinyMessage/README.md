EX\_25JAN - Mutiny message

Create a function named encodeMessage that receives 'message' as its parameter.

This function encodes a message by:

&nbsp;- Reversing the string.

&nbsp;- Replacing 'a' with '\*'.

&nbsp;- Replacing 'm' with '#'.

Parameter:

&nbsp;- message (String): The message consisting of lowercase alphabetical characters.

The function returns the encoded message as a String.
