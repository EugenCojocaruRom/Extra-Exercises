EX\_05MAR - Plump cushions simulator

Create a function named plumpCushions that receives numCushions and desiredPlumpness as its parameters.

It should return an array representing the state of each cushion after plumping.

Start with each cushion at a plumpness level of 1 and continue plumping until the desired plumpness level is reached.

Use loops to iterate through the cushions and increase their plumpness, and use conditional statements to describe the state of each cushion.

Parameters:

&nbsp;- numCushions (int): The number of cushions to plump (1 to 10 cushions).

&nbsp;- desiredPlumpness (int): The desired plumpness level (1 to 5, where 1 is barely plump and 5 is very plump).

The function returns a String array, where each element describes the final state of a cushion.

Use the following descriptions for each plumpness level:

&nbsp;- 1: "barely plump"

&nbsp;- 2: "slightly plump"

&nbsp;- 3: "moderately plump"

&nbsp;- 4: "very plump"

&nbsp;- 5: "extremely plump"

If the input parameters are outside the specified ranges, return an empty array.
